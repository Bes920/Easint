Hardened Security

ESSENTIAL
6 topics ▼

Privacy and Security go hand in hand. Establish a secure baseline before conducting any investigation.

🔐 Password Manager

Use a password manager to generate unique, complex credentials for every account. Also use to store TOTP codes and notes.

Bitwarden KeePassXC
💡 Don't keep all eggs in one basket — use a separate app for TOTP codes for better security
📱 MFA — Authenticator App

Multi-Factor Authentication is non-negotiable for account security.

Bitwarden Authenticator
🔑 MFA — Hardware Keys

Physical hardware keys provide the strongest MFA protection available.

YubiKey WebAuthn Test
🖥️ OS Hardening
Privacy.sexy O&O ShutUp10 (Win) BleachBit
🌐 DNS Privacy

Encrypted DNS prevents ISP tracking and DNS leaks.

NextDNS Cloudflare 1.1.1.1
💾 Full Disk Encryption

Enable FDE to prevent physical access to your data if device is stolen.

🍎 macOS System Preferences → Security & Privacy → FileVault → Turn On
🪟 Windows Control Panel → System & Security → BitLocker
🐧 Linux VeraCrypt or LUKS
💿 External VeraCrypt for all external drives

06

Sock Puppets

EXPERT
Account creation ▼

Sock puppet accounts are fake personas for undercover OSINT. Building them convincingly requires careful preparation.

✅ Prerequisites Before Creating Accounts
🎨 AI Image Generation Tools
This Person Doesn't Exist Random User Generated Photos Leonardo AI Unsplash AI Logo Maker
⚠️ Randomization is CRITICAL
❌ Reusing usernames — tools like whatsmyname.app scan 100s of websites to detect reuse. This WILL expose you.
  • Usernames → Generate via password manager
  • Passwords → Generate via password manager
  • Biography/Images → Use AI tools
  • Remove phone numbers or use VoIP only
📘 Facebook — Lessons Learned
❌ Account flagged within 5 minutes because:
1. Used well-known VPN → Use 5G/LTE or public WiFi
2. Used @duck.com email → Use Gmail, Outlook, or custom domain
3. No phone number → Provide real SIM, enable MFA, then remove
💡 Universal Tips for All Platforms
✅ Use 5G/LTE or hotel/coffee WiFi — never VPN during creation
✅ Use Gmail/Outlook/custom domain — not SimpleLogin or Duck
✅ Real SIM → enable MFA with app/Yubikey → delete phone from account

Metadata Management

ESSENTIAL
Removal & Insertion ▼

Metadata reveals hidden information in files. Learn to remove it for privacy and insert false data for disinformation.

🗑️ Metadata Removal

Most social media strips metadata — but not all. Remove before sharing:

  • Screenshot the image (strips EXIF)
  • Use offline tools (recommended)
ExifCleaner
💡 ExifCleaner works on PDF and Word files too

✏️ Metadata Insertion

Insert false metadata for disinformation or canary traps:

theXifer.net
💡 Can also modify metadata in Word/Excel document properties
🔗 Use the ExifTool in the main Easint Tools to instantly analyze file metadata
